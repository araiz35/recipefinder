 Interactive Recipe Finder

This Recipe Finder Web Page is a dynamic front-end application built with semantic HTML5 elements like `<header>`, `<nav>`, `<section>`, and `<footer>`The project features a robust search system that allows users to filter recipes by keyword or category using JavaScript DOM manipulation. 
 Key Features:
Interactive Elements:** Includes collapsible cooking instructions and a favorites list with duplicate validation
Visual Polish:** CSS transitions provide visual feedback and highlight recipes on hover
Technical Standards:** Developed for Assignment 02, the code emphasizes clean formatting, proper commenting, and efficient DOM traversal
